# QueroConhecer
> Aplicativo para desenvolvimento Swift/iOS

* Linguagem
    * Swift 5.5
    * MVC

* Auto Layout
    * Constraint
    * Assets

* Components
    * UIView
    * UILabel
    * UINavigation
    * UITableView
    * UIButton

    * SearchBar
    * AlertController
    * Extension Equatable

    * Enum
    * Gesture recognizers
    * Propriedades computadas
    * UserDefaults para persistência de dados
    * MapKit
    * MKAnnotation
    * ColorSet
    * Segue
